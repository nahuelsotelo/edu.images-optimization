Optimización manual

Guardar para web en Photoshop

  1. Elegir el formato adecuado
  2. JPG: Elegir el nivel de compresion (75-70 óptimo)
  3. PNG: Limitar la paleta de colores
  4. Opcional: Activar la opción de descarga progresiva (JPG) / entrelazada (PNG)
  5. Quitar la metadata (sino la necesitamos)

La descarga entrelazada / progresiva aumenta el peso del fichero aunque el usuario perciba una mejora en la performance

Aplicaciones de optimización

  • ImgOptim
  • ImageAlpha
